<h2>Features</h2>
<ul>
  <li>Random / Custom / Original <strong>MAC address</strong> control</li>
  <li>One-click <strong>VPN</strong> (Germany &amp; Canada servers)</li>
  <li>Pre-configured <strong>Proxychains</strong> (3 high-quality proxies)</li>
  <li><strong>Tor Browser</strong> + <code>torsocks</code> terminal support</li>
  <li><strong>I2P</strong> router installation &amp; console launcher</li>
  <li>Full network <strong>reset / recovery</strong> in one command</li>
  <li>Beautiful <strong>hacker-style interface</strong> + matrix-like startup animations</li>
</ul>

<h2>Installation (approx. 3 minutes)</h2>

<pre><code>
git clone https://github.com/PawanSandeepa99/Feenix-Anonimizer.git
cd Feenix-Anonimizer
chmod +x feenix_installer3.sh
sudo ./feenix_installer3.sh
</code></pre>

<p>
  <strong>Main script location after installation:</strong><br>

<h2>Usage</h2>

<h3>Launch</h3>
<pre><code>
sudo /Feenix_Anonimizer/feenix_V4.3.sh
</code></pre>

<h3>Main Menu Capabilities</h3>
<ul>
  <li>Change or restore MAC address</li>
  <li>Connect / disconnect VPN</li>
  <li>Enable or disable Proxychains</li>
  <li>Launch Tor Browser</li>
  <li>Enable <code>torsocks</code> for terminal traffic</li>
  <li>Install &amp; start I2P router console</li>
  <li>Fully reset network to original state</li>
</ul>

<h3>Recommended Usage Flow</h3>
<ol>
  <li>Change MAC address</li>
  <li>Connect VPN (optional)</li>
  <li>Enable Proxychains</li>
  <li>Launch Tor Browser or use <code>torsocks</code></li>
  <li>Perform tasks</li>
  <li>Use Network Reset before exit</li>
</ol>
